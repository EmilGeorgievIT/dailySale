# Running of project

Make sure you have Node.js installed.

```sh
$ git clone git@github.com:EmilGeorgievIT/dailySale.git # or clone your own fork
$ cd dailySale
$ cd server
$ npm install
$ npm start
$ cd client
$ npm install
$ npm start
```
