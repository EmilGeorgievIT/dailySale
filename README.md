# MEAN Stack dailySale 


# Stack of technologies


`Mongo DB`;
`Express JS`;
`React JS`;
`Node JS`;

# Running of project

Make sure you have Node.js installed.

$ git clone https://github.com/EmilGeorgievIT/dailySale.git # or clone your own fork
$ cd dailySale
$ cd server
$ npm install
$ npm start
$ cd client
$ npm install
$ npm start
